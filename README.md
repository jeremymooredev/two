# My Stash Mark Archive

This repository stores articles saved with the My Stash Mark browser extension.

## Structure

```
articles/
├── YYYY/
│   └── MM/
│       └── article-title-timestamp.md
```

Each article is stored as a markdown file with YAML frontmatter containing metadata.

## Extension

Get the browser extension: [My Stash Mark](https://www.mystashmark.com)
